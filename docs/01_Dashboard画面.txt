================================================================================
  Dashboard（装置稼働状況）画面
================================================================================

■ 画面概要
装置一覧を表示し、TYPE（装置タイプ）とLINE（ラインA/B）でフィルタリング
できる画面です。装置名をクリックすることで、その装置に対するWIPロット一覧
画面へ遷移します。

■ URL
/
または
/Index

■ 機能詳細

【フィルタ機能】
1. TYPEフィルタ（複数選択可）
   - チェックボックスで複数のTYPEを選択可能
   - 選択されたTYPEの装置のみ表示
   - 変更時に自動的にフォームを送信して画面更新

2. LINEフィルタ
   - ドロップダウンで単一選択
   - "すべて"、"A"、"B" などから選択
   - 変更時に自動的にフォームを送信して画面更新

3. フィルタークリア
   - すべてのフィルタをクリアしてデフォルト表示に戻る

4. Work Progress画面へ
   - 作業進捗ダッシュボードへ遷移

※ ヘッダーにはWork Progress画面へのリンクのみが表示されます
  （Dashboardリンクは削除されました）

【装置一覧表示】
- TYPEごとにカード形式でグループ化
- 各装置のNAME（装置名）とLINE（ライン）を表示
- 装置名はリンクになっており、クリックするとWIP Lot List画面へ遷移

■ データソース

【使用テーブル】
- DC_Eqps: 装置情報テーブル

【取得データ】
- フィルタ条件に基づいて装置リストを取得
- TYPE順、NAME順でソート
- 重複しないTYPEとLINEのリストを取得（フィルタ用）

■ 実装ファイル
- Pages/Index.cshtml
- Pages/Index.cshtml.cs (IndexModel)

■ コードロジック（Index.cshtml.cs）

【OnGetAsync メソッド】
1. TypeFilter と LineFilter をクエリパラメータから受け取る
2. DcEqps テーブルをクエリ
3. TypeFilter が指定されている場合、該当するTYPEのみフィルタ
4. LineFilter が指定されている場合、該当するLINEのみフィルタ
5. TYPEとNAMEでソートして結果を返す
6. フィルタ用のTYPEとLINEのリストを取得

【重要なプロパティ】
- Equipments: フィルタされた装置のリスト
- Types: フィルタ用の全TYPEリスト
- Lines: フィルタ用の全LINEリスト
- TypeFilter: 選択されたTYPEのリスト（複数可）
- LineFilter: 選択されたLINE（単一）

■ UI要素のID

すべての動的要素には一意のIDが付与されています：
- eqp-name-{Id}: 装置名のリンク
- eqp-line-{Id}: 装置のLINE表示
- type-{Type}: TYPEチェックボックス
- lineFilter: LINEドロップダウン

■ 遷移先

1. WIP Lot List画面
   - 装置名をクリックしたとき
   - URL: /WipLotList?eqpName={装置名}

2. Work Progress画面
   - "Work Progress画面へ"ボタンをクリックしたとき
   - URL: /WorkProgress
